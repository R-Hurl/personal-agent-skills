# Personal Agent Skills

A collection of Claude Code skills I find helpful in my day-to-day development workflow.

## Skills

### `feature-interview`
Acts like a senior technical PM — interviews you about a high-level feature idea, challenges your assumptions, asks the design and feasibility questions you might not think to ask yourself, and breaks the feature down into smaller sub-tasks ready to hand off one at a time.

### `requirements`
Takes a small, focused feature or change and produces a concise requirements document optimized for Claude Code's Plan mode. Best used with the sub-tasks produced by `feature-interview`.

### `dotnet-code-review`
Reviews uncommitted changes in a .NET/C# project for logic errors, anti-patterns, and non-idiomatic code, then writes a structured report to `codereview.md`.
